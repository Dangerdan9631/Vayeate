- Implement canonical domain contracts and shared types

  - Verify by integration tests simulating write failures and successful retries.
- Implement lock/mutability/versioning policy service

  - Verify by policy tests covering allowed and rejected mutations with expected diagnostics.
- Implement deletion and selection fallback policies
  - Implement policy-gated deletion checks for protected/in-use/locked constraints.

  - Verify by integration tests for success, rejection, and no-state-drift paths.
